# Rattrapage Javascript : Social Network

## Etapes

1. Vous pouvez, au choix :
    - Cloner ce dépôt
    - Télécharger l'archive Enonce_Rattrapage_Javascript.zip et le décompresser
    - Télécharger les fichiers individuellement
      
2. Suivez l'énoncé du rattrapage (il n'y a pas de différence entre le fichier docx et le pdf)

3. Créez un dépôt github public qui contiendra le résultat de votre travail

4. Envoyez le lien du dépôt github au formateur en suivant les modalités qu'il aura demandées
